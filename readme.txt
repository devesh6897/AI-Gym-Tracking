# AI Gym Movement Tracker

########### IMPORTANT NOTE ###################

The dataset is NOT included in this zip file because its size is more than 1 GB.
The model has already been trained and is available inside the models folder.
If you want to train the model yourself, you must first download the dataset separately and store it inside the DATA folder before running the training script.

LINK:https://mmfit.github.io/


## Train Model
python train_model.py 


### Running with Webcam
python exercise_classifier.py --webcam


### Running with a Video File
python exercise_classifier.py --video resources/file_name.mp4

### Running with a Image File
python exercise_classifier.py --image images/file_name.jpeg/jpg/png

## Supported Exercises
- Pushups
- Squats
